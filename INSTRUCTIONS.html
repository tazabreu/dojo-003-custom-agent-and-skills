Dojo #3 — a deeper dive on custom agents, skills and background agents

Goals of this Dojo
  1. Experiment with key primitives: instructions, custom agents, and skills
  2. Automate a recurring task — such as scaffolding web-app pages, API endpoints, or data-pipeline test structures
  3. Validate how these primitives improve background agents capabilities
What We'll Do Today
